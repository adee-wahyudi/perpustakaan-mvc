# Requirement

- XAMPP v3.2.4
- PHP Version 7.3.9

# Fitur Sistem Informasi Perpustakaan

- Login admin
- Create, read, update, delete data Anggota
- Create, read, update, delete data Buku
- Upload foto anggota
- Cetak kartu anggota
- Cetak Nota
- Transaksi peminjaman
- Transaksi pengembalian
- Print laporan data anggota
- Print laporan transaksi

# Konfigurasi Database dan URL

Untuk konfigurasi database dan URL dapat dilakukan pada file config.php 
yang berada di direktori perpustakaan-mvc\app\config\config.php

# Cara Menjalankan Aplikasi

- Pindahkan folder "perpustakaan-mvc" yang baru diekstrak ke dalam folder htdocs
  yang berada di direktori C:\xampp\htdocs
- Buat database dengan nama perpustakaan dan import perpustakaan.sql
- Untuk menjalankan aplikasi dapat menggunakan link berikut:

  http://localhost:8080/perpustakaan-mvc/public
